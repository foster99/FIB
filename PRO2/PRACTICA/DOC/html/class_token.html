Fa el tractament d'una string actualitzant la informació interna perquè funcionin bé les consultores.

Precondición
cert
Postcondición
be = cert si i només si s és un string vàlid que pot formar part d'una comanda; quan be = cert, s'ha creat un objecte Token sobre el qual es poden fer consultes
